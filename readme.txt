This WordPress Seize the Day Plugin display random quote on every page 

refresh.
 
== Description ==
 
Adds random motivational quote on each page refresh. It reminds you to live 

life to the fullest. That is what "Seize the Day" means. You do not need to 

add quotes, it already bags for you.

Place [seize-the-day] to display quotes in page/post.
 
Seize the Day Meaning: It's an exhortation to live life to the fullest, 

getting the most out of each individual day.
 
For more detailed meaning please visit: 

http://www.christianstudy.com/data/nt/philippians_c3_v7a.html

We hope to always motivate you via this plugin.
 
== Installation ==
 
1. Place the `seize-the-day.php` file in your `wp-content/plugins/ directory.
  
2. Activate the plugin through the 'Plugins' menu in WordPress. That's it.

3. Removing the plugin will remove the capability.
